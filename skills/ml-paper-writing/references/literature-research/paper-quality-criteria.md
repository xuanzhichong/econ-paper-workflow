# Economics Paper Quality Criteria

Use this to prioritize which papers deserve close reading.

## Dimensions

### 1. Identification Credibility

- what assumption does the result hinge on?
- how believable is that assumption?

### 2. Data Value

- is the data unique, comprehensive, or policy-relevant?
- does the data support the stated claim?

### 3. Contribution Precision

- is the paper clear about what is new?
- does it separate contribution from marketing?

### 4. Writing And Table Clarity

- can you trace the argument quickly?
- are tables and notes interpretable?

### 5. Relevance For The New Project

- does it use a comparable design?
- does it study a comparable setting?
- does it reveal a useful empirical pitfall or pattern?
