# Data and Code Disclosure

Use AEA-style replication expectations as the default standard unless the target journal says otherwise.

Useful policy pages:

- https://www.aeaweb.org/journals/policies/data-code
- https://www.aeaweb.org/journals/data
- https://www.aeaweb.org/journals/app/editorial-policy

## Replication Package Components

- README with environment and execution order
- do-files or scripts in run order
- description of raw inputs
- generated intermediate and final outputs, when redistributable
- notes on restricted or confidential data
- package dependencies and version assumptions

## Disclosure Text Should Cover

- whether data are public, licensed, or restricted
- how replication files reproduce tables and figures
- which steps require non-public data access
- who to contact or where to apply for access, if relevant
