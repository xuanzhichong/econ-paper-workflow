# Economics Writing Techniques

## Contribution Sentence Pattern

```text
This paper studies [question] using [data/design]. The key identifying variation comes from [source]. We find [headline result]. Relative to the existing literature, the paper contributes by [specific margin].
```

## Table Introduction Pattern

```text
Table X reports the baseline estimates. Column 1 shows [minimal specification]. Column 2 adds [controls/FE]. Column 3 introduces [design-relevant change]. The estimate remains [stable / attenuated / more precise], which is consistent with [interpretation].
```

## Cautious Mechanism Pattern

```text
These results are consistent with [mechanism], though they do not by themselves establish that channel conclusively.
```

## Robustness Transition Pattern

```text
We next examine whether the result is sensitive to [design concern]. The goal is not to search over specifications, but to test whether the headline estimate depends on a narrow modeling choice.
```
