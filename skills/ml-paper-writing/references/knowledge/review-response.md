# Economics Review-Response Knowledge

Use this as a companion to the response-writing skill.

Key principles:

- answer the referee's actual concern, not a weaker version
- improve the manuscript when the comment reveals unclear exposition
- say directly when a request cannot be satisfied
- map every major response to a manuscript location

Good response letters:

- acknowledge the point briefly
- answer substantively
- describe the change
- identify the new location
- keep tone professional and non-performative
