# Review Classification

Classify referee comments by the research concern they raise.

## Primary Categories

- `identification`
- `robustness`
- `mechanism`
- `external-validity`
- `data-construction`
- `exposition`
- `literature-positioning`
- `minor-edit`

## What Each Category Usually Implies

### identification

- revisit assumptions
- add diagnostics or falsification tests
- tighten language if the design supports less than currently claimed

### robustness

- add alternative samples, variable definitions, clustering, or windows

### mechanism

- add evidence or weaken causal interpretation if the mechanism is only suggestive

### data-construction

- document sample restrictions, merges, or variable coding

### exposition

- rewrite the manuscript, not just the letter

### literature-positioning

- revise framing, add missing citations, or narrow novelty claims
