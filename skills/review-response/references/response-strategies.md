# Response Strategies

Choose a response based on the substance of the referee point.

## 1. Accept and Revise

Use when the referee identifies a real weakness or omission.

Template:

- acknowledge the point
- say what changed
- name the new table, appendix, or section

## 2. Clarify Existing Logic

Use when the paper already contains the core idea, but the exposition failed.

Template:

- acknowledge the confusion
- state the intended logic
- say where the text was rewritten

## 3. Partially Accommodate

Use when the request is reasonable but only partly feasible.

Template:

- explain what was added
- explain what could not be done
- connect the partial change to the original concern

## 4. Decline with Justification

Use when the requested change would be inappropriate, impossible, or misleading.

Template:

- respectfully explain why
- note any clarification added to the manuscript
- avoid defensive or dismissive language

## Good Strategy Mapping

- identification concern -> new design defense, new check, or narrower claim
- robustness concern -> targeted sensitivity analysis
- mechanism concern -> added evidence or softened language
- exposition concern -> rewritten text plus signposting
