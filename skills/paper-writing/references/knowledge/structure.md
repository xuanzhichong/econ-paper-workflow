# Structure Patterns for Empirical Economics Papers

## Introduction Pattern

1. motivate the question
2. explain why current evidence is incomplete
3. state the design and data
4. summarize the main finding
5. position the paper in the literature

## Empirical Strategy Pattern

1. define treatment and outcome
2. explain identifying variation
3. present baseline specification
4. discuss threats and planned checks

## Results Pattern

1. baseline table
2. identifying diagnostics
3. mechanism or heterogeneity
4. robustness appendix

## Appendix Pattern

1. data construction detail
2. supplementary specifications
3. placebo or falsification tests
4. extra institutional detail
