# Empirical Economics Paper Structure

## Default Structure

1. Introduction
2. Institutional background / context
3. Data
4. Empirical strategy / identification
5. Main results
6. Mechanisms / heterogeneity
7. Robustness
8. Conclusion
9. Appendix

## Section Intent

### Introduction

- motivate the question
- state why it matters
- preview identification
- summarize the contribution relative to the literature

### Institutional Background

- explain the setting, policy, market, or rule change
- provide only the context needed to understand the design

### Data

- define the sample and core variables
- explain sources, merges, and restrictions

### Empirical Strategy

- present the estimating framework
- state assumptions and threats

### Results

- start with core estimates
- move supporting material to later sections or appendix

### Appendix

- house robustness variants, extra tables, and replication details
