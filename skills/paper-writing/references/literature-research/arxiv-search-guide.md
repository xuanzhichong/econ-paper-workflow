# Economics Literature Search Guide

This file keeps its old path for compatibility, but the active workflow is economics-first.

## Default Search Order

1. Google Scholar for broad discovery
2. RePEc / IDEAS for economics metadata and versions
3. NBER for working papers
4. AEA and field-journal sites for published versions

## Search Questions

- what is the exact research question?
- what identification strategies have already been used?
- what data sources recur in the literature?
- where do findings disagree?

## Minimum Extraction Fields

- citation metadata
- publication status
- identification strategy
- data source
- main finding
- limitations relevant to the new project
