# Economics and Agricultural Economics Writing Sources

Use these as the default source families for drafting, verification, and manuscript preparation.

This file is a source map for empirical economics and agricultural economics writing.  
Its purpose is to clarify **which source family should be used for which writing task**, rather than treating all sources as interchangeable.

A good writing workflow should distinguish between:

- discovery sources
- verification sources
- journal-policy sources
- model-writing sources

---

## 1. Literature Discovery

Use these sources to find relevant papers, map the literature, and identify related designs.

Preferred discovery sources:

- Google Scholar
- RePEc / IDEAS
- National Bureau of Economic Research
- AEA journals
- field journals relevant to the topic
- leading agricultural economics journals when relevant

Use these sources to answer questions such as:

- what is the exact research question?
- what identification strategies have been used?
- what data sources recur in the literature?
- where do findings disagree?
- which papers are directly relevant versus only background-related?

### Working rule

- use `Google Scholar` for broad discovery
- use `RePEc / IDEAS`, `NBER`, and journal sites for cleaner metadata, version tracking, and related-paper tracing
- do not stop at discovery; move toward structured extraction and verification

---

## 2. Citation Verification

Use these sources to verify metadata, versions, and claim-source alignment.

Preferred verification sources:

- DOI landing page
- publisher pages
- Crossref
- RePEc / IDEAS
- NBER working-paper pages

Use these sources to check:

- author list
- year
- title
- journal venue or working-paper series
- DOI if published
- working-paper number if unpublished
- whether a later published version exists
- whether the cited claim actually matches the source

### Working rule

- verify the version before citing
- prefer the published version when it exists and is substantively the right source
- use at least two independent sources when the citation is central to the paper’s contribution, identification argument, or factual claim

---

## 3. Submission and Editorial Policy

Use these sources when preparing the manuscript for advisor circulation, submission, revision, or resubmission.

Preferred policy sources:

- target journal author guidelines
- journal editorial policy
- data and code policy
- formatting or style guide
- submission-system instructions when relevant
- AI or disclosure policy if relevant

Use these sources to determine:

- manuscript structure and formatting rules
- abstract, highlights, appendix, or significance-statement requirements
- data and code availability expectations
- reference style and submission file requirements
- revision-letter or resubmission expectations when applicable

### Working rule

- always verify journal-specific requirements from official journal sources
- do not rely on memory or secondhand summaries when preparing final files for submission

---

## 4. Writing Inputs

Use these sources as models for structure, tone, and empirical writing patterns.

Preferred writing inputs:

- well-written published papers in the target literature
- top field-journal introductions and empirical strategy sections
- target-journal papers with similar designs
- clear robustness and appendix examples
- strong regression tables and figure examples
- referee response letters from economics journal contexts when available

Use these inputs to learn:

- how to frame the contribution
- how to describe identifying variation
- how to write results around tables
- how to present robustness without clutter
- how to keep mechanism evidence appropriately cautious
- how to structure a professional response letter

### Working rule

- prefer papers that are not only topically relevant, but also strong in design clarity and writing discipline
- use these as pattern sources, not boilerplate to copy

---

## 5. Source Priority by Task

### For literature review

Prefer:

1. discovery sources
2. metadata/version sources
3. published papers in relevant journals

### For citation verification

Prefer:

1. DOI or publisher page
2. Crossref
3. RePEc / IDEAS / NBER
4. Google Scholar only as a discovery aid

### For paper drafting

Prefer:

1. strong published papers in the target literature
2. target-journal articles
3. well-structured empirical strategy and results sections
4. appendix and response-letter exemplars when needed

### For submission preparation

Prefer:

1. official journal guidelines
2. editorial and data/code policy
3. style and formatting instructions
4. submission-system instructions

---

## Working Rule

A disciplined paper-writing workflow should always know:

1. where ideas are discovered  
2. where citations are verified  
3. where submission rules are confirmed  
4. where writing patterns are learned  

If all four source families are mixed together without distinction, the writing process becomes less reliable, less reproducible, and more likely to produce citation or submission errors.
