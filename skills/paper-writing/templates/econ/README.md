# Economics Template Entry Points

Use these templates as planning and structure guides for empirical economics manuscripts.

## Included Entry Points

- `working-paper-outline.md`
- `journal-submission-outline.md`
- `appendix-outline.md`
- `response-letter-outline.md`

## Default Usage

- working draft or seminar paper -> `working-paper-outline.md`
- journal revision or field-journal target -> `journal-submission-outline.md`
- appendix planning -> `appendix-outline.md`
- referee response package -> `response-letter-outline.md`

These are workflow scaffolds, not publisher-owned style files.
